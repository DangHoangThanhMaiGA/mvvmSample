# MvvmSample Overview
This app uses new MVVM Architecture and intergrates Android Components.

# Library

* Gradle version: `3.1.3`
* Kotlin version: `1.2.3`
* Build tool: `27.0.3`

* Lifecycle: `1.1.1`
* Room persistence: `1.1.0`
* Retrofit: `2.3.0`
* Paging: `1.0.0`
